## This project was made with:

* [Python 3.7](https://www.python.org/) 

## how to run the project?

* Create a virtualenv with Python 3.
* Activate virtualenv.
* Install dependencies.
* Run migrations.

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python py_main.py
```
